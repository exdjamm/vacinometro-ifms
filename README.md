## 🚀 Configurando o ambiente
Para ter um cópia local do projeto, siga estas etapas:

### ☑️ Pré-requisitos
- Node.js instalado

	- https://nodejs.org/en/download/

### ⬇️ Instale as dependências necessárias
```sh
npm install
```

### ⚛️ Inicie o servidor de desenvolvimento
```sh
npm start
```
